# 🎵 每日唱片

> [README-en](README-en.md) & [在线页面](https://forestlet.github.io/everyday-album/)

## 简述

1. 本项目旨在创建一个“每日唱片”网站，每个人都可以找到某天，特别是 ta 生日当天发行的唱片 🥳

2. 唱片 JSON 格式

   ```json
   {
     "title": "xxx", // 唱片名
     "artist": "xxx", // 作者
     "release_time": "xxxx-xx-xx", // 发行时间
     "filename": "xxx.jpg" // 封面文件名
   }
   ```
